# AETHER

## Russian

Эта страница создана для обновления лаунчера по майнкрафту Aether. Поддержите разработку, написав предложения, что можно добавить, нахождение багов и т.п. 
Надеюсь на звезду для продвижения =)


Как же скачать лаунчер? Можете воспользоваться автоустановщиком (aether_installer.exe). Или же скачать .zip последней релизнутой версии и внутри запустить Aether.exe.

## English

This page is created to update the Aether Minecraft launcher. Support the development by writing suggestions on what can be added, finding bugs, etc. 
Hope for a star to promote =)


How to download the launcher? You can use the auto-installer (aether_installer.exe). Or download the .zip of the latest released version and run Aether.exe inside.
