# Aether Launcher

Aether Launcher - лаунчер Minecraft с контейнерами, установкой контента с Modrinth, автообновлениями и отдельным хранилищем данных.

## Ссылки

- GitHub: https://github.com/Sqikpeek/Aether
- Releases: https://github.com/Sqikpeek/Aether/releases
- Скачать установщик: откройте последнюю версию в Releases и скачайте `Aether Installer.exe`
- Скачать пакет обновления: в последнем релизе используется `Aether Update.zip`

## Возможности

- Отдельные контейнеры для разных версий Minecraft и наборов модов.
- Поддержка Vanilla, Fabric, Quilt, Forge и NeoForge.
- Поиск и установка модов, модпаков, датапаков и плагинов через Modrinth.
- Общие миры между контейнерами.
- Автоподбор Java и настройка памяти.
- Копирование логов игры для диагностики.
- Автообновления лаунчера через GitHub Releases.
- Темы интерфейса и мгновенная смена языка.

## Установка

1. Откройте страницу Releases.
2. Скачайте `Aether Installer.exe` из последнего релиза.
3. Запустите установщик и выберите папку хранилища.
4. Установщик скачает `Aether Update.zip`, распакует лаунчер и создаст ярлык.

## Для релизов

В каждый новый GitHub Release добавляйте два файла:

- `Aether Installer.exe` - установщик.
- `Aether Update.zip` - пакет лаунчера для установки и автообновления.

Установщик и обновлялка выбирают именно `Aether Update.zip` и не используют `Aether Installer.exe` как пакет обновления.
